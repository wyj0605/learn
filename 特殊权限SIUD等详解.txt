特殊权限
SUID 运行某程序时，相应进程的属主是程序文件自身的属主，而不是启动者

[root@localhost pcre-8.34]# ls -l /usr/bin/passwd
-rwsr-xr-x 1 root root 22984 2007-01-07 /usr/bin/passwd
	chmod u-s file
	chmod u+s file
	如果FILE本身原来就有执行杼，则SUID显示为s;否则显示S

	chmod u+s /bin/cat
SGID 运行某程序时，相应进程的属组是程序文件自身的属组，而不是启动都所属的基本组
	chmod g+s file
	chmod g-s file

Sticky 在一个公共目录，每个都可以创建文件，删除自己的文件，但不能删除别人的文件
	chmod o+t DIR
	chmod o-t DIR
chmod 1755 /bak/test


FACL : Filesystem Access Contronl List
	利用文件扩展保存额外的访问控制权限

setfacl 设置facl
getfacl 获得facl

[root@localhost shsh]# getfacl sed.txt
# file: sed.txt
# owner: root
# group: root
user::rw-
group::r--
other::r--

setfacl -m g:aaa:rwx

owner---> group ----> other

owner --> facl ,user --> group --> facl,group -->other


setfacl 
	-m:设定
		u:username:perm
		g:groupname:perm
		u 代表用户
		g 代表组
	setfacl -m u:hadoop:rw inittab

	-x :取消

	setfacl -x g:mygroup inittab
